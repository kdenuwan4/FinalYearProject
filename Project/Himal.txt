LoRa-Based Long Range Remote Sensor Node: Use LoRa modules (RYLR896) to transmit sensor data over 2–5km without cellular data.

Li-Fi (Visible Light Communication) Audio/Data Bridge: Using a Power LED as a transmitter and a photodiode/LDR as a receiver to send data through light.

Handheld RF Signal Strength Meter: An ESP32-based tool that scans WiFi/Bluetooth spectrums and displays a heat map of signal congestion.

DTMF Controlled Industrial Appliance Switcher: Control machinery by sending specific audio tones over a standard phone call (useful for remote areas with no internet).

Morse Code / Digital Text Encoder-Decoder: A device that takes keyboard input, encodes it into a custom protocol, and transmits it via 433MHz RF modules.

Packet Loss Monitor for Wireless Links: Two MCUs communicating via NRF24L01 modules to calculate and display real-time Signal-to-Noise Ratio (SNR) and Bit Error Rate (BER).

RFID Attendance with Cloud Sync: A system that logs ID cards to local memory and syncs to a Google Sheet via WiFi when a connection is available.

Dynamic Frequency Hopping Spread Spectrum (FHSS) Demo: Two MCUs that switch communication channels every 500ms in a synchronized pattern to avoid interference/jamming.
